# Auto Installer for Node.js Applications

## Download & Install

### Direct Download (Windows)

[Download latest version](https://github.com/readloud/auto-installer/releases/latest/download/auto-installer.exe)

[Latest release](https://github.com/readloud/auto-installer/releases/latest/download/auto-installer.exe)

[Specific version](https://github.com/readloud/auto-installer/releases/download/v1.0.0/auto-installer.exe)

[GitHub Packages](https://npm.pkg.github.com/@readloud/auto-installer)

### Using PowerShell

```powershell
# PowerShell
Invoke-WebRequest -Uri "https://github.com/readloud/auto-installer/releases/download/v1.0.0/auto-installer.exe" -OutFile "auto-installer.exe"
./auto-installer.exe
```

### Using npm

```bash
npm install -g @readloud/auto-installer
auto-installer
```

### Via NPM

```bash
npm install -g @readloud/auto-installer
auto-installer
```

### Via script downloader

```bash
node download-installer.js
```
